# Outline v0.8（Station 0 / ch01：塑封童鞋之门）

## 本章目标
- 用第一人称把“门 -> 副本 -> 走错 -> 贴脸 -> 归位 -> 回声”跑通。
- 读者读完不需要懂全机制，但要觉得：这事像真的发生过。

## 事故类型骨架（虚构化）
- 老旧居民楼楼梯间火灾 + 拥堵（不指向任何真实案件）。

## 开场（现实：失物招领处）
- 我深夜在失物招领处地下仓库清点“无主逾期遗物”。
- 遗物：塑封儿童鞋，编号缺一位（063-?），塑封里有水汽；鞋底有玻璃粉。
- 失真介质：热敏纸监控截帧（楼梯间门口，门缝纯黑）；背面手写“下行疏散，勿向上”。

## 门出现（副本入口）
- 我盯着门缝那片黑看久了，仓库灯抽搐。
- 货架背后出现一扇不该存在的铁门，门缝里也是那种纯黑。
- 我推门进去。

## 副本（楼梯间火灾现场）
- 热先到，焦甜味后到；扶手烫手；楼上脚步砸下来。
- 证据冲突：绿灯安全出口箭头指向上，但墙上/牌子/照片背面都提示向下。
- 我选错一次，楼梯折叠回原点。

## 贴脸（证词缝合体）
- 它拿着对讲机，说的是“保持冷静、有序撤离、不要拥挤”这类正常口令。
- 口令的尾音被剪过，像多人拼接。
- 它逼我报出遗物编号，逼我把“我是谁/我来干嘛”说成结论。
- 它不会给答案，只会让“正确方向看起来像陷阱”。

## 缺失物证（本章核心）
- 我在消防栓箱看到“本该有水带/接头”的位置空了。
- 我用现场痕迹（拖痕、水渍、金属反光、胶带味）推到杂物间/垃圾袋附近。
- 找到水带接头：黄铜发黑，外圈有牙印咬痕。

## 归位与出口
- 我把接头拧回接口，动作写细（涩、咬合、震动）。
- 声音突然对齐；出现“新漆铁门”。
- 证词缝合体在平台上模仿动作，多做一步“回拉”，像引我把接头再拔下来。
- 我不解释，直接推门出站。

## 现实回声与钩子
- 我回到仓库，手里真的多了接头；塑封袋上出现新指纹。
- 我尝试拍照留证，照片褪成空白/0KB，留不住。
- 陌生语音留言：“你把不该归位的东西，归位了。”
- 章末再落一个更私人、更盗墓笔记式的钩子：我手掌纹路里有玻璃粉，洗不掉。
