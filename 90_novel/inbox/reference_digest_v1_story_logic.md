# Reference Digest v1（结构抽样：开篇/中段/收束 -> 可执行方法）

目标：不是“读完全文背梗概”，而是把 reference 里最值钱的叙事方法抽出来，变成 Station 0 可复用的写作动作（不抄句子、不复刻独特细节、不影射真实案）。

说明：本 digest 采用“结构抽样”方式阅读（开篇 1 段 + 典型推进 1 段 + 收束/回声 1 段），用于提炼叙事规律。长篇全本逐字通读成本极高，后续按需要补抽样。

## 1) 《盗墓笔记》：真实感不是词藻，是“走位+物件+判断会犯错”
- 开篇抓钩：开场直接给一个“具体异常”（比如血从土里渗出来），读者立刻知道“这事不对劲”，不需要解释世界观。
- 代入感来源：
  - 我怎么走、手摸到什么、看见什么细节（锈、潮、泥、味道）比“我很害怕”有效。
  - 对话像人：粗话、互呛、拿捏，不端着讲设定。
  - 判断会错：我有职业经验，但不神；我做的每次判断都会付代价。
- 用在 Station 0：
  - 入口遗物第一段就给“可以画出来的异常”（味道、磨损、缺一位编号）。
  - 不要解释“门机制”，先写“我怎么确认它不对”。

## 2) 《诡舍》：无限流的核心不是规则，是“空间压迫+老手信息差”
- 开篇抓钩：大巴没司机，极快建立“现实逻辑断裂”，同时给群像（7人）让恐惧可传播。
- 推进方式：
  - 老手不一次性讲完，而是用“你走错就死”的口吻逼你跟着做。
  - 场景压迫比鬼更吓人：雾、别墅、开门/关门、每一步都像踩进坑。
- 用在 Station 0：
  - 一章里至少出现 1 个“地图能画出来”的压迫结构（狭窄走道、循环站台、错位出口）。
  - 让“老手/旁观者”只给够用的信息，别写成教程。

## 3) 《死亡万花筒》：门感=入口异常+组队信息差+回到现实的回声
- 开篇抓钩：日常失衡 -> 推门 -> 熟悉楼道变长走廊 -> 多扇铁门，读者直接进入“门感”。
- 推进方式：
  - 新人困惑很具体（我不知道这是哪、怎么出去），老手的冷静带节奏。
  - 过门不是“看完一段恐怖”，而是“做完一件必须做的事”才回到现实。
  - 回声点让读者相信“这事没结束”。
- 用在 Station 0：
  - 单章必须闭环（进入->推进->升级->解决->回归->回声），不要写成“进去看一眼就回”。
  - 回声点要具体且难摆脱（味道、痕迹、0KB、突然新增的一行字）。

## 4) 《全球高考》：信息披露节奏很硬，开场即压力
- 开场就把人按进“必须过关”的情境，压力来源清晰（不及格就死）。
- 场景选择直接把题目和环境绑死，读者不会问“为什么在这”，只会问“怎么活”。
- 用在 Station 0：
  - 每一站必须有“我必须做的动作”，且动作能被验证（归位/打孔/封口）。
  - 信息披露按“能活->能推进->能理解”的顺序给。

## 5) 《末日乐园》：日常场景切入，快速埋下“人要杀你”的现实恐怖
- 用很生活的场景（麦当劳、可乐、汉堡）铺底，让“恐怖”落在现实里。
- 恐怖不是怪物，是“你身边的人不对劲”。
- 用在 Station 0：
  - 入口现实段别写玄：先把“我”的职业日常写稳，异常才更刺。
  - 恐怖优先写“人做的事”（掩盖、装正常、清理、威胁）。

## 6) 《惊悚乐园》《轮回乐园》：强目标+强动作驱动（但要避免写成系统播报）
- 优点：目标清晰、动作连续、节奏稳，不会在恐怖点前后塌陷。
- 风险：一旦写成“系统提示/机制说明”，AI味与教程味会暴涨。
- 用在 Station 0：
  - 只学“目标推动事件”，不要学“系统播报语气”。
  - 任何机制必须被改写成“我在现场里怎么发现、怎么被逼着做”。

## 7) 《从姑获鸟开始》：世界观材料的正确用法
- 设定可以深，但不要抢戏。设定只在“推动选择/解释后果”时出现。
- 用在 Station 0：
  - 圣经/台账写在后台，正文只写我当下能感知到的那一点。

## Station 0：把上述方法落到“可执行动作”

### A) 单章闭环（强制）
1) 现实入口：遗物 + 失真介质（一个具体异常）
2) 门出现：位置不合理但物理细节真实
3) 进入后：第一恐怖点（画面）+ 第一个错误判断（代价小）
4) 升级：人性/凶案压迫逼我站位（代价变大）
5) 解决动作：找回并归位缺失物证（动作具体、可验证）
6) 回归：出口出现（不白给）+ 回声点落地 + 钩子（下一站）

### B) 读者不懂就会骂的 3 类硬伤（写完自查）
- 物理逻辑断裂：门/锁/链/钥匙/位置互相打架。
- 因果跳步：我突然“知道了”、突然“就这么做了”，中间缺一拍。
- 恐怖不来自故事：堆术语、讲机制、讲规则墙，读者不怕只烦。
