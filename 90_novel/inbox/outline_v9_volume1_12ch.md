# 卷一总纲（v0.9 / 12章）

主线之谜（可验证）：
- Q1（1-3章推进）：失物招领处为什么会收到“带介质”的遗物？是谁投递？
- Q2（4-8章推进）：编号缺失（房号/批次/时间）是否有规律？缺失的“最后一位”代表什么？
- Q3（9-12章收束）：每次“归位”为什么会让现实叙述更干净？谁在利用我完成“锁门/封口/封条”？

三章小拐点：
- 第3章：第一次出现“同一批遗物的内部联系”（不是巧合）
- 第6章：第一次确定“编号缺失规律”的一条可验证规则
- 第9章：第一次反推“投递者的操作路径”
- 第12章：卷末揭露一层更深的组织/系统结构，留下更大门

## ch01-ch03 细纲（摘要）
### ch01《门链》
- 6拍：遗物+便签 -> 门出现 -> 6楼走廊错位 -> 正常求助逼开门 -> 找回门链归位 -> 猫眼确认+出口+回声点

### ch02《猫眼》
- 6拍：门牌号遗物 -> 门出现 -> 公寓走廊门牌乱序 -> 门外站人不敲门 -> 找回猫眼归位 -> 现实猫眼变模糊

### ch03《冰柜票根》
- 6拍：小票遗物 -> 门出现 -> 小卖部后间 -> 冰柜“喘气的空” -> 找回旋钮帽归位 -> 现实冰箱结霜异常

备注：每章正文写作必须落地为“走位+物件+代价”，不写规则墙，不写抽象术语恐怖。
