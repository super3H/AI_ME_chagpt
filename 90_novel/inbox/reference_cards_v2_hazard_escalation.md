# Reference Cards v2（“凶险感”怎么写出来：从诡舍/死亡万花筒抽动作）

目标：解决你说的“太简单、凶险写不出来”的问题。这里不讲设定，不抄原句，只抽能直接用在正文里的写法。

## 1) 让读者先紧张：把“危险”写成倒计时与稀缺
- 不是“我很害怕”，而是：时间不够了、灯快灭了、门马上开、车要到站。
- 稀缺：工具/光源/钥匙/出口只有一个，且会被夺走或失效。
- 用法：开场 300-800 字内出现一个“可验证的紧迫物”（滴答/灯闪/五分钟/呼吸起雾）。

## 2) 让人觉得会死：用“可逆的小代价”铺路，再上“不可逆的大代价”
- 小代价：摔倒、划伤、手被夹、鞋底打滑、手机没信号。
- 大代价：同伴当场出事、出口被封、路线被剪断、你被迫留下证据或背锅。
- 用法：每 800-1200 字升级一次代价，且升级必须来自“一个选择或一个动作”。

## 3) 群体恐惧比鬼更吓人：让人先崩溃
- 让一个人先情绪崩盘（哭、砸东西、质问、逃跑），读者马上知道“这次不一样”。
- 主角不用讲规则，只要写：大家怎么互相看、怎么避开责任、怎么推你去做事。
- 用法：中段安排一次“群体压迫会议”，把主角逼到必须站位的位置。

## 4) “老手”不能像 NPC 发任务：信息要带血
- 老手说话要像经历过：有回避、有缺口、有不愿提起的细节。
- 老手只说“足够你活下去的一句话”，剩下让主角自己从现场验证出来。
- 用法：老手台词避免诗化总结，改成“具体提醒 + 个人创伤反应”（比如手抖、避开某个位置）。

## 5) 悬疑不是顿悟：写“观察 -> 怀疑 -> 验证”
- 禁用句式：我突然意识到/我终于明白/我忽然明白过来（除非前面有足够证据链）。
- 用法：每个推理至少 3 拍：
  - 观察（物理细节）
  - 怀疑（提出 2-3 个可能）
  - 验证（用动作去试）

## 6) 险境要“干活”：危险发生时，主角必须同时做两件事
- 例：一边扛东西一边辨路；一边撬锁一边躲人；一边撒谎一边找物证。
- 这样读者会觉得：不是你讲得好，是你真的在赌命。
- 用法：高潮段强制“双任务并行”，并写清楚手、脚、呼吸的负荷。

## 7) 恐怖要落在“人做的事”
- 现场的恐怖要指向：遮掩、清理、封口、造假、推责、围观。
- 用法：给一个“封口动作”（锁门/贴胶带/擦掉字/撕走尾段），让读者明白这不是天灾。

## 8) 破局要“成本”与“后遗症”
- 通关不是按按钮，是付出：伤、损失一件东西、背上一个新的麻烦。
- 回声点要能长期折磨你：味道回流、手上印子、文字新增、证据 0KB 但在“占用空间”。

## Station 0 直接落地（写作自检）
- 本章紧迫物：有没有在 800 字内出现（倒计时/灯灭/门要开）？
- 本章代价升级：至少 3 次（小->中->大），且都来自选择。
- 推理链：每个“结论”前是否写了观察与验证？
- 高潮段：是否“双任务并行”？是否写清楚身体负荷？
- 破局成本：通关后有没有留下真实的损失与后遗症？
