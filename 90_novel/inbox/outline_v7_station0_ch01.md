# Outline v0.7 (Station 0 / ch01)

## 章节目标
- 把“事故遗物 -> 现场缝隙 -> 证据冲突 -> 怪物贴脸 -> 缺失物证归位 -> 现实回声”完整跑一遍。
- 读完这一章，读者不需要懂完整机制，但要相信：这是现实质感的现场，不是规则闯关。

## 事故类型骨架（虚构化）
- 老旧居民楼楼梯间火灾 + 人群拥堵（不指向任何真实案件）。

## 开场（遗物入口）
- 林硕深夜在失物招领处仓库帮忙清点一批“无主遗物”。
- 一只塑封的儿童鞋（或一次性雨衣口袋里的钥匙串），编号贴纸缺一位。
- 同批材料里有一段监控截帧：画面停在楼梯间门口，门缝里是黑。

## 入站（现场缝隙）
- 他把塑封撕开，鞋底灰像还湿。
- 仓库的灯一闪，空气里先到的是热，再到的是甜腻的焦味。
- 再抬头：他站在一段狭窄楼梯间，墙皮起泡，扶手烫手，楼上有脚步砸下来。

## 证据冲突（误判触发）
- 他看到两条“叙述”同时存在：
  - 门上的安全出口箭头指向上。
  - 墙上广播牌却写“下行疏散”。
- 他先按常识往下走，结果楼梯像被拼错：转了两层回到原地。
- 这一段不讲规则，只用“走不出去”的物理感制造压迫。

## 怪物第一次贴脸（证词缝合体）
- 楼道里有人影一闪，像拿着对讲机。
- 它开口说的话不是恐吓，而是几句很正常的“指挥口令”，但每句话都像从不同人的喉咙里挤出来。
- 林硕意识到：这些话和他在材料里见过的“口供模板句”完全一致。
- 它不追，他一停，它就站得更近一点，像在等他把结论说出口。

## 缺失物证（本章核心）
- 林硕在楼梯拐角看到一个消防栓箱，门半开，里面没有水带。
- 现场材料里本应出现的“水带接头”在任何截图/日志里都没有。
- 他最终在一堆被踩扁的塑料袋里摸到一个冰冷的金属接头，外圈有牙印一样的咬痕。

## 归位与出口
- 他把接头拧回消防栓接口。
- 周围的声音突然对齐：广播、脚步、喘息都落在同一个节奏上。
- 楼梯间尽头出现一扇之前不存在的铁门，门上的漆是新的。
- 证词缝合体站在楼梯平台上，像在“复述”他刚才做的动作，但复述里多了一步，试图引他回头。
- 林硕不解释，直接推门出去。

## 现实回声（章末钩子）
- 他回到仓库，手里却真的多了那枚金属接头。
- 更糟：仓库里那只儿童鞋的塑封袋上，出现了新的污渍指纹，像有人刚摸过。
- 手机响起一条陌生号码的语音留言，只有几秒对讲机杂音，然后有人用很平静的声音说：
  - “你把不该归位的东西，归位了。”

## 伏笔
- 牙印咬痕：不像人咬的。
- “新漆铁门”：有人在现场里修过路。
- 留言者：可能是寄遗物的人，也可能是另一个“走出来的人”。
