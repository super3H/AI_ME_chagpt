# 小说应用快速调用示例

把下面这段直接发给执行模型即可：

```text
按“小说应用执行提示词”严格执行，不要反问；先 Top3 打分，再自动选最高分生成分章大纲与样章。
题材方向：近未来悬疑科幻 / 无限流（可写成更偏现实的都市惊悚）
目标读者：18-35 岁，偏好高概念与人物反转
篇幅目标：长篇连载
叙事视角：第三人称（可在关键章切换第一人称日记/录音）
节奏要求：中速偏高密度
风格偏好：冷峻、克制、信息密度高、规则感强
禁忌项：不写纯玄学解释；不写无动机反转；不强行“系统送挂”

提示：如果你觉得某个方向“不吸引/不恐怖”，优先提高“沉浸场景可写性”（可走的空间+可触摸的物件+可见的机制/代价），避免写成纯流程/纯术语。
```
