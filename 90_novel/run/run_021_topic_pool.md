## Top 8 选题池
1) 《塑封童鞋》：老旧居民楼楼梯间火灾

- 缺失物证：消防栓水带接头（材料里消失）
- 恐怖点：广播指示互相打架；证词缝合体用“正常指挥口令”把人引向错误楼层
- 主线钩子：现场出现“新漆铁门”，像有人提前改过路
- 分数：92

2) 《对讲机回放》：地下停车场一氧化碳/缺氧事故

- 缺失物证：封堵用的胶带卷芯（现场照片里被抹掉）
- 恐怖点：墙里传来“按住别动”的口令；越安静越听见有人在你耳边数数
- 主线钩子：回放里出现一个不在记录名单里的频道代号
- 分数：89

3) 《断电电梯卡》：商场停电踩踏
- 遗物：断裂的电梯卡套（卡面照片被刮花）+ 广播录音（中间有空拍）
- 缺失物证：应急照明的电池盒（现场清单缺项）
- 恐怖点：黑暗里人群的“重复句”像潮水；证词缝合体只会说一句“别回头”
- 主线钩子：电池盒上有同一种牙印咬痕
- 分数：88

4) 《无名工牌》：工地密闭空间窒息
- 遗物：无名工牌（姓名栏空白，但磨损像戴过很多年）
- 缺失物证：通风管道的检修钥匙
- 恐怖点：空间极窄，呼吸声变得像别人的；你越想喊人，嗓子越像被粉尘糊住
- 主线钩子：工牌背面有“失物处”同款编号格式
- 分数：86

5) 《碎屏旧手机》：出租屋煤气爆燃
- 遗物：碎屏手机（最后一条短信只有半句）
- 缺失物证：燃气阀门的旋钮帽（照片里被擦掉）
- 恐怖点：证词缝合体会替你把短信补完；补完的每个字都像在替你签字
- 主线钩子：短信收件人是一个不存在的号码段
- 分数：85

6) 《双份票根》：长途大巴夜间侧翻

- 恐怖点：车厢像被折成两段；窗外永远是同一棵树
- 主线钩子：浅票根背面有“已归位”的红字
- 分数：83

7) 《带盐味的雨衣》：堤岸坍塌落水
- 遗物：一次性雨衣（内侧有盐味结晶）
- 缺失物证：救生绳的绳扣
- 恐怖点：水声从干燥的地面里冒出来；证词缝合体站在你身后，像潮湿的呼吸
- 主线钩子：绳扣有被同类工具剪断的痕迹
- 分数：81

8) 《消失的钥匙串》：老旧旅馆夜间火灾
- 遗物：房卡钥匙串（房号缺一位）+ 焦黑的楼层示意图

- 主线钩子：示意图角落有“施工完成”印章
