# run_018（Station 0 / ch01）流水线分发包

目标：产出《第 1 章：终点站不认人》的可发布正文（v1-clean），并完成三模型评鉴与读者报告。

约束（最高优先级）：
- 取材只允许写“现实案件类型结构”（地铁火灾/踩踏事故），细节必须彻底虚构化；不得出现真实案件名称/真实地名/真实人物。
- 恐怖来自空间、物件、机制、代价；禁止作者解释“为什么吓人”。
- 每个证据点必须至少触发一次：微危险 或 微选择 或 微代价。
- 需要出现一次“无凭断言惩罚”（电流/窒息等即时惩罚）。

本章结构卡（6拍）：
1) 误入：末班车后，站台忽然“换皮”，闸机不认人；倒计时开始。
2) 规则撞脸：手机/广播要求提交“出站证明”；角色下意识想用“系统故障”解释，触发轻惩罚。
3) 证据点1（同号票）：捡到两张同编号票据互相冲突；取证时被闸机前的无脸轮廓“盯住”。
4) 证据点2/3（监控黑10秒 + 日志缺页）：进入监控室/站务间，发现关键10秒黑屏、日志缺页断口整齐；时间被吃掉。

章末钩子：回到现实，门禁/身份出现“验证中/合规分字段”。
