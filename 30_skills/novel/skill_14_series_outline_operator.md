# Skill 14: 长篇连载“整体大纲”算子（案件恐怖版 / 第一人称）

目的：把 reference 里的可复用优点抽成一套“你自己的大纲方法”，后续写作严格跟大纲走，避免一章一个想法导致跑偏。

本 skill 只学结构与手法，不复写任何 reference 原句，不影射真实案件。

## 参考吸收（只学方法）
- 《盗墓笔记》：
  - 真实感来自“我”的现场记录感：走位清楚、物件带线索、判断会犯错。
  - 大纲不是设定罗列，而是“线索 -> 行动 -> 危机 -> 出路 -> 反转/钩子”的连续链。
- 《死亡万花筒》：
  - 门=章节入口；新人困惑、老手信息差推动节奏；每章必须交付一条可验证信息。
- 《诡舍》：
  - 空间压迫是核心舞台：地图能画出来；每次推进都付代价；出口永远不白给。

## 输入
- 系列圣经（Series Bible）
- 案件类型池（Case Bank，全虚构）
- 产能与体量：每周产出/单章字数/一卷章数
- 叙事要求：第一人称；恐怖必须故事驱动（人做的事 + 压抑现场）
- 禁区：不写真实案名/地名/人物；不写规则墙；不靠抽象术语恐怖

## 输出
1) 【卷结构蓝图】（建议 1 卷 12 章）
2) 【12 章总纲表】（每章一行，带主线线索推进）
3) 【ch01-ch03 细纲】（可直接写正文）
4) 【一致性台账字段】（人物状态/资源/伏笔/案件类型边界）

## 执行步骤
### Step 1: 定“主线之谜”（必须可验证，不是玄学空话）
写成 3 个问题：
- Q1（近期，3 章内要推进）：比如“失物为什么回流到招领处？”
- Q2（中期，6-8 章推进）：比如“编号缺失规律是什么？”
- Q3（长期，12 章收束）：比如“谁在用‘归位’改写现实叙述？”

每章至少推进：回答 1 个小问题，或否定 1 个错误假设，或新增 1 个可验证线索。

### Step 2: 用“案件类型骨架”排章（案件不是目的，是载体）
从 Case Bank 里选 12 个案件类型，要求：
- 4 个“密闭空间压迫”（窒息/车库/管廊/电梯/停尸间）
- 4 个“室内凶案压抑”（旅馆/合租/厨房/卫生间）
- 2 个“人群与公共场所”（夜市/商场/车站）
- 2 个“伪安全机构”（医院/殡仪馆/营业厅）

避免连续 3 章同皮肤（比如连续都是出租屋）。

### Step 2.5（可选）: 从“真实案件”抽骨架，再脱敏改编
只学逻辑链条与物证链，不复写细节，不影射可识别的真实事件：
- 抽骨架（6项）：入口物件、空间皮肤、关键人物关系、关键物证、关键失误/清理痕迹、最后的“封口动作”。
- 脱敏（至少改 3 项）：地点、人物关系、职业/动机、时间/天气、现场物件。
- 防影射：优先把两个来源骨架拼接（A 的动机 + B 的物证/空间），避免一一对应。
- 写作呈现：不要在正文里声明“改编自某案”，只让读者感觉“像真的”。

### Step 3: 每章用同一张“章卡模板”
每章必须填：
- 案件类型（全虚构）
- 入口遗物（可摸、可带回）
- 失真介质（便签/截帧/录音碎片）
- 副本皮肤（门后场景）
- 关键恐怖点（故事驱动，写成“发生了什么”而不是“概念是什么”）
- 缺失物证（小而关键）
- 通关动作（必须具体动作：装回/贴回/塞回/锁上/签字）
- 回声点（带回现实的污染/后果）
- 主线线索（本章交付 1 条）
- 章末钩子（下一章读者想知道什么）

### Step 4: 写“12 章总纲表”（强约束）
输出一个表格：
- 章名
- 入口遗物
- 案件类型
- 缺失物证
- 本章交付线索
- 章末钩子

并写明：每 3 章一个小拐点（揭露一个层级更深的事实）。

### Step 5: ch01-ch03 细纲（给写手直接写）
每章 6 拍结构（必须可写）：
1) 现实入口（遗物+介质）
2) 门出现
3) 进入现场 + 第一恐怖点
4) 人性/凶案压力升级（逼“我”站位）
5) 找回并归位缺失物证（动作写细）
6) 出口 + 回声点 + 钩子

### Step 6: 一致性台账字段（给审计用）
- 我（林硕）的职业能力边界：能看细节，但不会凭空推理超能力
- 性格底色：爱自我合理化，但在不确定时会后退半步
- 道德底线：不主动伤害无辜
- 资源：工具/打火机/手套/手机状态
- 伏笔：每条伏笔必须写回收章号（计划）

## 输出模板（直接复制）

```md
- 三章小拐点：3/6/9/12章各揭露一层

- 场景物件清单：...

【一致性台账字段】
- 人物状态：...
- 资源：...
- 伏笔：...
```
