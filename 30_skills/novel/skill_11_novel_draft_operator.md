# Skill 11: 小说分章与正文草稿算子

## 目标
将 `skill_10` 最高分方向转化为可持续写作的分章结构和正文草稿。

## 输入
- 来自 `skill_10` 的入选方向
- 目标篇幅（字数区间）
- 叙事视角（第一/第三人称，单线/双线）
- 节奏要求（慢热/中速/高密度）
- 风格偏好（冷峻/轻快/抒情/克制等）

## 执行步骤
1. 先输出完整分章大纲（建议 8-12 章）。
2. 逐章标注：
- 章节目标
- 关键事件
- 情绪曲线
- 伏笔与回收点
3. 生成样章（第1章或用户指定章节）。
4. 输出连续创作所需的“角色状态表 + 设定一致性约束”。

## 输出格式
```md
作品名（暂定）：
一句话卖点：
目标篇幅：
叙事视角：

【分章大纲】
第1章：
- 章节目标：
- 关键事件：
- 情绪曲线：
- 伏笔：

...

【样章正文】
（不少于1200字，按用户要求可扩展）

【连续写作约束】
- 角色状态表：
- 世界观硬约束：
- 文风约束：
- 禁忌与边界：
```

## 强制约束
- 正文必须服务主线冲突，不堆设定。
- 伏笔要有预期回收路径。
- 人物行为必须符合既有动机和角色状态。
