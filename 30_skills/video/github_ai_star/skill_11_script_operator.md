# Skill 11: GitHub AI 高星项目脚本算子

## 目标
将最高分开源项目选题转成 8-15 分钟长视频脚本，突出“为何值得关注、怎么上手、如何避坑”。

## 输入
- 来自 `skill_10` 的最高分选题
- 目标时长（8-15 分钟）
- 目标受众
- 关键证据节点（至少 5 条）

## 脚本骨架
1. 冷开场：这个项目为何突然值得关注。
2. 第1章 项目定位与问题定义。
3. 第2章 核心能力拆解。
4. 第3章 快速上手路径。
5. 第4章 与同类项目对比。
6. 第5章 边界与踩坑提醒。
7. 结尾 + CTA。
