# Skill 11: 双时间线串联脚本算子（Jackalope 风格）

## 目标
将最高分选题转成 8-15 分钟长视频脚本，突出 A/B 双线并行到机制汇合。

## 参考文献
- 默认参考：`30_skills/video/jackalope_bridge/references/jackalope_case.md`
- 用于校准叙事节奏：传闻层 -> 证据层 -> 机制层 -> 现实层。

## 输入
- 来自 `skill_10` 的最高分选题
- 目标时长（8-15 分钟）
- 表达风格（理性拆解 / 叙事考据）
- 关键证据节点（至少 5 条）

## 脚本骨架
1. 冷开场：提出“为何两件事看似无关却有关”。
2. 第1章 A线：异常/传闻/偶然起点。
3. 第2章 B线：主流演进主线。
4. 第3章 汇合机制：解释连通路径。
5. 第4章 机制放大与验证。
6. 第5章 现实意义与边界。
7. 结尾 + CTA。

## 质量检查
- 开场 40 秒内明确反差问题。
- A/B 两线完整且汇合清晰。
- 至少包含 5 条可核验事实。
- 若引用 Jackalope 案例，只能借鉴结构，不得直接复写内容。
