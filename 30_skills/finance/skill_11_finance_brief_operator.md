# Finance Brief Operator

## 目标
基于 `skill_10_finance_focus_operator.md` 的结果，产出一份更适合日常使用的金融分身简报。

## 输出格式
```md
# 金融分身简报

## 结论

## 市场主线

## 跟踪标的 / 方向

## 风险与失效条件

## 接下来 48 小时怎么做
```

## 执行要求
1. 先给结论，不先铺背景。
2. 每条判断都尽量挂到可观测变量上。
3. 区分“事实”“解释”“动作”。
4. 动作建议必须能执行，例如：
   - 继续跟踪
   - 等待数据
   - 缩小范围
   - 回避噪音
5. 如果输入是零散的，也要先整理成结构再输出。

## 默认风格
- 高密度
- 少空话
- 不做投资承诺
- 更像研究员工作台，而不是媒体稿
