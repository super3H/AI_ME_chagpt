# Finance Quick Start

当任务属于金融分身，默认按下面方式执行：

1. 收集输入：
   - 用户文本判断
   - 图片摘录
   - 关注标的 / 行业 / 宏观关键词
   - 最新财经新闻
2. 执行：
   - `30_skills/finance/skill_10_finance_focus_operator.md`
   - `30_skills/finance/skill_11_finance_brief_operator.md`
3. 输出：
   - 结论
   - 市场主线
   - 重点跟踪
   - 风险边界
   - 48 小时动作

如果用户直接点名 `40_projects/finance_analysis_app/`，则同时读取该项目的 `PROJECT_MANIFEST.md`。
